Etiquetas escenciales de HTML

¿Para qué sirve !DOCTYPE html?
-indica al navegador que el documento está escrito en HTML5
-No tiene atributo solo indica que el documento es HTML5.

¿Para qué sirve html?
-sirve como una instrucción para el navegador sobre cómo mostrar un elemento en una página web, actuando como un marcador que define la estructura y el formato del contenido
-Su atributo lang → define el idioma del contenido. html lang="es"

¿Para qué sirve head?
-sirve como un contenedor para los metadatos del documento web, es decir, información sobre la página que no se muestra directamente al usuario.
-No tiene atributos “obligatorios” usuales, pero dentro se incluyen etiquetas con atributos como meta charset="UTF-8". codificación que puede representar casi todos los caracteres de todos los idiomas, incluyendo acentos, símbolos, emojis, etc.

¿Para qué sirve body?
-Es el contenedor principal de todo lo que se ve en una página web. Todo lo que quieres que el usuario vea en el navegador como texto, imágenes, enlaces, formularios, tablas y otros elementos interactivos debe ir dentro de esta etiqueta.
-No tiene atributos obligatorios, pero puede tener:
-class → para estilos CSS.
-id → para identificar el cuerpo.
-style → para estilos en línea.

¿Para qué sirve h1, h2, ..., h6?
-se utilizan para definir encabezados y subtítulos en un documento web, estableciendo una jerarquía de importancia y organización del contenido.
-Sus atributos son:
-class → aplicar estilos CSS.
-id → identificar el encabezado.
-style → cambiar estilo directamente.

¿Para qué sirve p?
-Al encerrar el texto entre las etiquetas indica al navegador que ese bloque de texto debe ser tratado como un párrafo, lo que generalmente implica un espacio en blanco antes y después de cada uno, facilitando la lectura.
-Sus atributos son:
-class → aplicar estilos CSS.
-id → identificar el párrafo.
-style → estilos en línea.

¿Para qué sirve a?
-se utiliza para crear enlaces hipertexto, permitiendo a los usuarios navegar entre diferentes páginas web.
-Sus atributos son:
-href → URL a la que apunta el enlace.
-title → texto que aparece al pasar el mouse.

¿Para qué sirve img?
-Esta etiqueta permite agregar visualmente contenido que enriquece la experiencia del usuario y mejora la comprensión de la información.
-Sus atributos son:
-src → ruta de la imagen.
-alt → texto alternativo.
-width / height → tamaño de la imagen.

¿Para qué sirve ol?
-crea listas ordenadas, donde los elementos se muestran con una numeración secuencial, ya sea por números, letras o símbolos.

¿Para qué sirve ul?
-Agrupa elementos relacionados que no requieren una disposición numérica o alfabética, como una lista de compras etc.

¿Para qué sirve li?
-representa un elemento individual dentro de una lista, ya sea ordenada o no ordenada y solamente se utila dentro de las etiquetas ol y ul como en los ejemplos de arriba.
